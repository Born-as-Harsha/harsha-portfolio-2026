## 📥 Installation & Setup

Clone the repository:

```
git clone https://github.com/Born-as-Harsha/harsha-portfolio.git
```

Install dependencies:

```
npm install
```

Run locally:

```
npm run dev
```

Build for production:

```
npm run build
```
